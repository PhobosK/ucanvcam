Installing ucanvcam

You can download ucanvcam in compiled form from http://www.makesweet.com

Here are the basic instructions for compiling and installing ucanvcam:

  • Make sure you have CMake installed.
  • Download ucanvcam
  • Run "./configure"
  • Run "make"
  • ucanvcam should now be available in the "build" directory.
